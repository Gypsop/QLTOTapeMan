# 阶段 6：UI 复刻（Qt Widgets / .ui）规划

目标：用 Qt Designer .ui 复刻原“直接读写”体验及配置器入口，保持独立文件管理器/预览对话框，全部 Widgets，无 QML。本阶段仅文档。

## 参考源（原项目）
- `LTFSConfigurator.vb`：主界面（驱动列表、装载/退带、挂载/映射、调试面板、进入 LTFSWriter 按钮）。
- `LTFSWriter.vb`：直接读写主窗口（树/列表文件区、状态栏、工具栏/菜单、状态灯、日志、进度、控制按钮）。
- `FileBrowser.vb`：独立树形选择器对话框（全选/按大小/正则筛选、勾选联动、剪贴板可选）。
- `TapeCopy.vb`：块复制对话框（可作为后续“工具”页参考）。

## 窗口/对话框设计
1) **MainWindow (Configurator)**
   - 区域：
     - 设备列表 (ListView) + 刷新/自动刷新
     - 驱动信息文本框、设备路径输入框
     - 按钮：装载(螺纹/非螺纹)、退带、装载/卸载、挂载/解除、映射盘符、进入 LTFSWriter、调试面板开关
     - 可选隐藏的 FUSE/Changer/Remount 按钮（按需右键显示）
     - 调试面板：SCSI 直发、LogSense 导出、输出文本框
   - 状态：底部消息框/日志视图。

2) **LTFSWriterWindow (Direct RW)**
   - 左树 + 右列表：展示 ltfsindex 目录/文件，支持复选、排序、筛选。
   - 顶部工具栏/菜单：
     - 文件：加载索引、保存索引、自动导出
     - 操作：开始/暂停/继续/停止/Flush/Eject，更新数据区索引
     - 视图：显示文件数/过滤；语言切换入口（菜单触发翻译）
   - 控制区：
     - 离线/在线切换；额外分区；块长；哈希开关；限速；索引写入间隔；容量刷新间隔；自动清洁；加密密钥/密码（输入框+设置按钮）；条码显示
   - 状态栏：
     - 状态灯 (LWStatus)、进度条、速度/剩余时间、当前位置、容量
   - 日志区：多行文本；可复制；错误高亮（按需）。

3) **FileBrowserDialog**
   - 树形视图（复选）
   - 按钮：全选、按大小、正则筛选、确定/取消
   - 选项：是否同步到剪贴板。

4) **附加对话框/面板**（预留）
   - TapeCopy 工具页/对话框（块复制）
   - Debug SCSI 发送对话框（可借调试面板）

## 信号/交互要点
- 设备列表选中 -> 更新信息/按钮可用性，与 `SelectedIndex` 逻辑一致。
- 按钮状态：根据驱动是否挂载/装载，在线/离线模式，写入进行中（禁用相关控件）。
- 文件区复选联动：与 `FileBrowser.vb` 的递归勾选/三态逻辑一致。
- 进度/状态灯：由 `LtfsService` 回调驱动；任务栏进度可省略或使用 Qt/OS API 代替。

## 布局/风格
- 采用 Qt Designer 生成 .ui，逻辑在 .cpp。
- 翻译：使用 Qt 翻译文件（zh, en, zh_Hant）；菜单/标签文本可从 Resx 对应到 Qt 翻译。
- 无 QML，无自绘复杂控件，尽量使用标准 Widgets（QTreeView/QListView/QTableView/QProgressBar/QStatusBar/QPlainTextEdit）。

## 资源
- 图标/字符串：后续阶段可从原资源抽取（`res/`、`*.resx`），转换为 Qt 资源 `.qrc`。

## 验证
- 在无设备环境下提供 stub 数据：虚拟设备列表、模拟索引树，便于 UI 调试。
