# QLTOTapeMan

QLTOTapeMan is a cross-platform LTFS management tool for LTO tape drives, built with Qt 6 and C++.

## Goal
To provide a professional, easy-to-use GUI (and Terminal GUI) for personal users to manage LTO tapes reliably.

## Requirements
- Qt >= 6.0
- C++17 compliant compiler
- CMake >= 3.16

## Build Instructions
```bash
mkdir build
cd build
cmake ..
cmake --build .
```
